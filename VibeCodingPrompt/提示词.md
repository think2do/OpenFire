# 项目背景
我准备做一个可以和AI聊天的工具，主要呢，是服务于普通的小白用户，他们可以在手机上和AI模型发起对话。

# 产品功能
1. 用户可以通过聊天窗口和模型发起多轮聊天对话；
2. 模型会结合上下文响应用户对话内容；
3. 利用模型结合前两轮自动为对话生成对话主题，默认为新对话；
4. App可以在本地保存用户的每一组对话记录；
5. 用户可以查看已保存的消息，并继续与模型发起对话；
6. 准许用户删除历史对话组。

# 页面布局
“对话页”
1. 页面最上方为工具栏 左侧为“历史对话”按钮，中间“对话标题”，右侧为“新对话”按钮；
2. 工具栏下方区域为“对话展示区域”，展示用户和模型的对话内容；
3. 最下方为“文本输入框”、右侧为“发送”按钮。

“历史对话页”
1. 页面最上方为工具栏 左侧为“返回按钮”，中间为页面标题“历史对话”，右侧为“新对话”按钮；
2. 工具栏下方为“对话记录块”组成的历史“消息列表”；
3. “对话记录块”展示对话命名，与最新一条消息还有生成时间。

# 交互逻辑
1. 用户首次打开App进入“对话页”；
2. 用户可点击“文本输入框”调起键盘后键入内容；
3. 用户可“发送”按钮后将输入内容发送至模型；
4. 模型会根据用户的问题生成回复，并在“对话展示区域”进行展示，最新的消息在下方；
5. 用户点击“新对话”按钮后可以开启一个新的“对话页”；
6. 用户点击“历史对话”可以跳转至“历史对话页”；
7. 用户点击“返回按钮”时可返回至上一页面；
8. 用户点击“对话记录块”后跳转至已存储消息的“对话页”。

# 输出要求
1. 利用Xcode进行App开发，主要适配 14Promax；
2. 所用到的模型可以使用“资源”中提供的内容；
3. 自行测试调整。
